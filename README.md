# Meals App

A React-based application for browsing and managing meal recipes.

## Features

- Browse meals by category
- Search for specific meals
- View detailed meal information
- Save favorite meals
- Responsive design

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

Runs the app in development mode at [http://localhost:5173]

## Project Structure

```
src/
├── components/
├── pages/
├── App.js
└── index.js
```

## Technologies

- React
- CSS3
- JavaScript ES6+
